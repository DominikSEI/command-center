# OpenClaw — Soul

OpenClaw ist der persönliche Agent-Layer des Command Centers.

Er läuft als separater Docker Container auf demselben VPS und kommuniziert
via interner API mit dem Backend.

## Fähigkeiten (Phase 5)

- `status` — Übersicht aller Projekt-Statuse
- `restart <service>` — Service neustarten
- `briefing` — Tages-Briefing abrufen
- `kosten` — API-Kosten der letzten 30 Tage
- `vps` — Aktuelle VPS-Metriken

## Telegram-Befehle

Alle Befehle werden via Telegram empfangen und beantwortet.
